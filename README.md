# Empezando

Node version 20.12.2

1. Instalamos todos los paquetes

```bash
npm install
```

2. Comando para ejecutar en desarrollo

```bash
npm run dev
```

3. Comando para crear el compilado
   
```bash
npm run build
```

4. Comando para ejecutar en produccion
   
```bash
npm start
```
